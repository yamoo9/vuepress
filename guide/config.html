# .vuepress 디렉토리

VuePress 환경설정을 위한 디렉토리 .vuepress/를 docs/ 디렉토리 내부에 생성합니다.

$ mkdir docs/.vuepress
1

NOTE

.vuepress/ 디렉토리 내부에는 VuePress 사이트를 구성하는 특별한 파일들이 배치됩니다.

# config.js 파일

VuePress 사이트를 구성하는데 필요한 파일 docs/.vuepress/config.js를 만든 후 아래 코드를 작성합니다.

// docs/.vuepress/config.js

module.exports = {
  title: '헬로! VuePress',
  description: 'VuePress 러닝 가이드북과 함께해요!'
};

# 구성 디렉토리

이어서 .vuepress 디렉토리 내부에, 다음 디렉토리를 생성합니다.

디렉토리 설명
components Vue 컴포넌트 관리 디렉토리 (글로벌 컴포넌트로 자동 등록)
styles 스타일 파일(.styl) 관리 디렉토리
public 정적 리소스(images, css, js 등) 관리 디렉토리

 
 
 


.vuepress/
├── components/
├── styles/
├── public/
└── config.js
