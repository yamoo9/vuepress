# VuePress란?

VuePress는 Vue 프로젝트 문서와 같은 기술 문서 제작에 최적화된 도구로 싱글 페이지 애플리케이션(SPA) 사이트 제작에 사용됩니다. 아래 문서는 모두 VuePress로 제작되었습니다. VuePress로 제작된 사이트는 정적 HTML 페이지로 구성되며 검색 엔진에 최적화(SEO) 된 페이지를 제공합니다.

  • vuejs.org
  • router.vuejs.org
  • vuex.vuejs.org
  • cli.vuejs.org
  • vue-loader.vuejs.org
  • ssr.vuejs.org
  • vue-test-utils.vuejs.org
  • rollup-plugin-vue.vuejs.org

# 작동 원리

VuePress는 Vue, Vue Router 및 Webpack으로 구성된 싱글 페이지 애플리케이션 제작 도구입니다. Vue.js를 사용해본 경험이 있다면 손쉽게 사용할 수 있습니다. (Vue DevTools를 사용해 디버깅 할 수 있습니다).

VuePress 빌드(build) 명령을 수행하면 서버 사이드 렌더링(SSR) 애플리케이션을 만들고 가상으로 각 패스(path)를 방문해 정적 HTML을 렌더링합니다. 이러한 방법은 Nuxt.js, Gatsby 같은 프로젝트에서 영감을 얻었습니다.

Markdown 파일은 markdown-it을 사용하여 HTML로 컴파일 된 다음, Vue 컴포넌트 템플릿으로 처리됩니다. 뿐만 아니라 Markdown 파일에서 Vue 코드를 직접 사용할 수 있으며 이 방법은 동적으로 콘텐츠를 포함해야 할 때 유용합니다.
