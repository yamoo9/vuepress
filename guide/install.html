# 설치

# 글로벌 설치

글로벌 설치 명령은 모든 프로젝트에서 vuepress 명령을 사용해 VuePress를 설치해 사용할 수 있습니다.

 


$ yarn global add vuepress@next
# 또는 npm i -g vuepress@next
1
2

# 로컬 설치

특정 프로젝트에서만 VuePress를 사용하고자 한다면, 로컬 설치해 사용합니다.

 


$ yarn add vuepress@next -D
# 또는 npm i vuepress@next -D
1
2

NOTE

Webpack 3.x 의존 모듈을 사용하는 기존 프로젝트에 VuePress를 설치해 사용할 경우, NPM 대신 Yarn 패키지 매니저 사용을 권장합니다. 이런 경우 NPM은 올바른 종속성 트리를 생성하지 못하기 때문입니다.
